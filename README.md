# Медицинский центр «Здоровье»

Сайт медицинской клиники с онлайн-записью к врачу.

## Что это за проект

Простой сайт для медицинского центра. Пациент может посмотреть список
врачей, услуги и записаться на приём онлайн. Данные хранятся в Supabase.

## Структура проекта

| Файл / папка | Что внутри |
|--------------|------------|
| `Новое/index.html` | Главная страница сайта |
| `Новое/doctors.html` | Страница со списком врачей |
| `Новое/services.html` | Страница с услугами клиники |
| `supabase/functions/symptom-search/` | Серверная функция Supabase (поиск по симптомам) |

## Как запустить

Открой файл `Новое/index.html` двойным щелчком — он откроется в браузере.

## Технологии

- HTML / CSS — внешний вид и страницы сайта
- Supabase — база данных и серверные функции
