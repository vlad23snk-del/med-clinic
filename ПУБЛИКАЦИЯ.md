# 📣 Инструкция: как опубликовать сайт и собрать отзывы

Этот файл — пошаговый чек-лист для новичка. Делайте сверху вниз, ничего не пропуская.

---

## Шаг 1. Загрузить изменения на GitHub

Откройте терминал в папке проекта и выполните по одной команде:

```bash
git add .
git commit -m "Подготовка к публичному релизу: README, лицензия, скриншоты, Vercel"
git push
```

После этого README со скриншотами будет виден на странице репозитория:
👉 https://github.com/vlad23snk-del/med-clinic

---

## Шаг 2. Опубликовать сайт на Vercel (бесплатно)

1. Зайдите на **https://vercel.com** и нажмите **Sign Up → Continue with GitHub**.
2. Разрешите Vercel доступ к вашим репозиториям.
3. Нажмите **Add New… → Project**.
4. Найдите репозиторий **med-clinic** и нажмите **Import**.
5. В поле **Root Directory** нажмите **Edit** и выберите папку **`docs`**. ← это самый важный шаг!
6. **Framework Preset** оставьте `Other`. Поля **Build Command** и **Output Directory** не трогайте.
7. Нажмите **Deploy** и подождите ~1 минуту.
8. Готово! Vercel покажет ссылку вида `https://med-clinic-xxxx.vercel.app` — это ваш сайт в интернете.

> 💡 Скопируйте эту ссылку и вставьте её в начало файла `README.md` (строка «Демо (Vercel)»),
> затем снова сделайте `git add . && git commit -m "Добавил ссылку Vercel" && git push`.
